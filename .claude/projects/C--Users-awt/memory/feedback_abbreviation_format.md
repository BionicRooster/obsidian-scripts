When using an abbreviation in a vault note, first write the full text followed by the abbreviation in parentheses, then use only the abbreviation for the rest of the note.

Example: "Toyota Production System (TPS)" on first use, then "TPS" thereafter.

**Why:** User's explicit formatting rule for vault notes — abbreviations must be self-defining on first use.

**How to apply:** Applies to all vault notes written by Claude. On first occurrence of any abbreviation, expand it fully. Never open a note with a bare abbreviation.
