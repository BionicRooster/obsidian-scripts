When writing any vault note that includes foreign language words or phrases, define them inline using parentheses immediately after the term on first use.

Format: **Term** (native script if different — "English translation")

Examples from Japanese:
- **Omoiyari** (思いやり — "consideration for others")
- **Sōji** (掃除 — the daily cleaning practice)
- **Satoyama** (里山 — "village mountain")

For languages using the same alphabet (French, Spanish, Latin, German, etc.), omit the script field:
- **Verein** ("association" — German formal civic organization)

**Why:** User explicitly confirmed this format as "useful clarifying presentation" after reviewing the Japanese Community Improvement Culture note (2026-07-03). Applies to all foreign language terms in all vault notes.

**How to apply:** Applies to any note Claude writes containing non-English words — research summaries, synthesis pages, ingested sources, recipe notes with foreign terms, Bahá'í notes with Arabic/Persian terms, etc. Define on first use; subsequent uses may use the term alone.
