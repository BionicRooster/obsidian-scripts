On dark blue or blue-gray backgrounds, blue-tinted text must be very bright (near-white blue) to be readable. Blues require the most contrast of any color for this user.

**Why:** Wayne has difficulty distinguishing low-contrast blue text on dark backgrounds.

**How to apply:** When choosing text colors for dark-themed UIs (dashboards, cards, etc.):
- Accent blues used for body text must be `#9fc8ff` or brighter — never a mid-tone like `#6aa3ff` for readable text
- Secondary/muted text in blue-gray tones must be `#b0b8d8` or lighter
- Prefer off-white (`#eceffe`) for primary readable text rather than any blue tint
- Reserve medium blues for decorative elements (borders, backgrounds) only, not text
