Add every newly encountered name to the People Index immediately, with a wikilink to the note where they appear. No threshold for People Index entry.

Do NOT create a file in `15 - People\<Name>.md` until a person appears in 5 or more vault notes. At that threshold, create a full biography note — not a stub.

**Why:** The People Index is a running catalog of everyone in the vault. The 15-People folder is reserved for people who are genuinely significant (5+ links). Stubs in 15-People create clutter; the People Index handles discovery until the threshold is reached.

**How to apply:** At the end of classify workflows, add all new names to the People Index with a link to the source note. Only when a person crosses 5 vault links should a full biography note be created in `15 - People\<Name>.md`.

**Insertion rule:** Always insert the new entry in its correct alphabetical position within the section — never append to the end of a section. The People Index is alphabetically ordered; appending breaks the sort order.
