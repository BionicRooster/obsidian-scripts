When converting email/blog posts into recipe format, **always retain the narrative/story text** from the original file.

**How to apply:**
- Place narrative content in a `## Story` section **after** the recipe (after Notes, before Related Notes)
- This applies even if the narrative preceded the recipe in the original file
- Separate paragraphs with blank lines
- Never discard author commentary, anecdotes, or blog post text — only reorganize it

**Why:** User corrected a conversion that discarded the blog narrative. Story context is part of the recipe's value.
